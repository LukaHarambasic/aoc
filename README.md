# aoc

## Installation

```bash
bun install
```

## Development

### Run

```bash
bun run start
bun run start:watch
```

### Test

```bash
bun run test
bun run test:watch
```

This project was created using `bun init` in bun v1.0.13. [Bun](https://bun.sh) is a fast all-in-one JavaScript runtime.
